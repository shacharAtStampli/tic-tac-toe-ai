# Tic Tac Toe - Smart AI Battle

A full-stack Tic Tac Toe game featuring intelligent AI agents.

## Project

The game is located in the `TicTakToe/` folder with:
- **Backend**: Java Spring Boot REST API
- **Frontend**: React single-page application

See [TicTakToe/README.md](TicTakToe/README.md) for detailed documentation.

## Quick Start

### Start Backend
```bash
cd TicTakToe/backend
mvn spring-boot:run
```

### Start Frontend
```bash
cd TicTakToe/frontend
npm install
npm start
```

Open http://localhost:3000 in your browser.

## Features
- Dynamic board size (3x3 to 10x10)
- Smart AI agents with strategic thinking
- 0, 1, or 2 human players
- Early draw detection
- Fair random first player selection
